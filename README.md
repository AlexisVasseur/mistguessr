# Silksong Mistguessr

A guessing game built with Vue.js where players identify mist locations from Hollow Knight images.

## Features

- Random image display from the mists collection
- Autocomplete input for guessing mist names
- Score tracking across attempts
- Clean UI styled with Hollow Knight color palette

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deploy to Vercel

1. Push this project to a Git repository
2. Import the project in Vercel
3. Vercel will automatically detect the Vue.js framework and deploy

## Structure

- `/public/mists/` - Contains all mist images
- `/src/App.vue` - Main game component
